Project 1

In this presentation we will show you a program compatible with a form that we will also show below, to facilitate the management of a computer technician in charge of devices in an institute. The form would be designed for anyone who detects an error or failure in any of the systems, who through the survey could notify the technician, informing him about the basic information so that it can be organized.

Program Description

The program is a web-based application built with HTML, CSS, and JavaScript, designed to manage incident reports from users in the institute. It includes a database (simulated with local storage) to store reports, allowing the technician to view, prioritize, and resolve issues efficiently.

Incident Report Form

Below is the form that users can fill out to report technical issues. It collects essential information to help the technician diagnose and fix problems quickly.
