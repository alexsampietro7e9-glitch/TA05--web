Project 1

Gestión de incidencias mediante formulario inteligente

Sistema de Gestión de Incidencias

Este proyecto presenta un sistema diseñado para facilitar el trabajo de un técnico informático dentro de un instituto. A través de un formulario claro e intuitivo, cualquier usuario puede reportar fallos o incidencias indicando el tipo de dispositivo, la ubicación y una descripción del problema. El objetivo es transformar un proceso caótico en un flujo de trabajo estructurado y fácil de gestionar.

Toda la información recogida permite al técnico organizar las incidencias por prioridad, tipo de error y zona del centro, mejorando la eficiencia y reduciendo el tiempo de respuesta. Este sistema está pensado para ser escalable, intuitivo y adaptable a cualquier entorno educativo o empresarial.
